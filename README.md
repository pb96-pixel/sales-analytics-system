\# Sales Analytics System



This repository contains an end-to-end Sales Analytics System built using Python as part of the Module 3 Assignment.



\## Project Overview

The project reads raw sales data, cleans and validates it, performs analytics, enriches it using an external API, and generates a detailed sales report.



\## Project Structure

\- Q1: File handling and data cleaning

\- Q2: Data preprocessing and validation

\- Q3: Data processing and analytics

\- Q4: API integration and data enrichment

\- Q5: Report generation

\- Q6: Main application execution



\## Features

\- UTF-8 safe file reading

\- Data cleaning and validation

\- Sales analytics (region-wise, product-wise, date-wise)

\- API integration using DummyJSON

\- Automated report generation

\- Interactive command-line application
